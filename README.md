## Requirements

To run the notebooks, you need to have the following Python packages installed:

```
ipykernel==6.29.5
numpy==2.2.2
pandas==2.2.3
scikit-learn==1.6.1
matplotlib==3.10.0
seaborn==0.13.2
```

You can install the required packages using the following command:

```bash
pip install -r requirements.txt
```
